###  Currently Building: Scalable E-Commerce Platform

I am currently working on a microservices-based e-commerce backend system that aims to simulate real-world, production-ready architectures.

The system will include:

* **API Gateway** for centralized routing & security
* **Auth Service** with JWT-based authentication
* **Product, Order & Payment Services** for core business logic
* **Notification Service** for event-driven communication
* Communictaion by using **REST + event-driven architecture (Kafka)**
* Fully containerized infrastructure using Docker

####  What I'm focusing on

* Designing **independent microservices with clear responsibilities**
* Implementing **secure authentication flows (JWT)**
* Building **scalable system architecture**
* Learning **service-to-service communication (REST & events)**
* Working with **Docker-based environments and orchestration**

####  Current State

* Microservice structure fully set up
* Each service is an independent Spring Boot application
* Each service has:

  * its own **Dockerfile**
  * its own **.env configuration**
  * its own **database**
* Central **docker-compose setup** for the entire system
* API Gateway already integrated as entry point
